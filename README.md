# KeluLinuxKit
a tools to setup your dev environment on Debian. Now we are developing on the way, be careful and don't try it for your working environment.

## Waht's in it
* personal .bashrc
* Maximum Awesome for Linux
* tmux-powerline
* iptables
* mutt & msmtp
* PPTP
* dropbox
* transmission
* xrdp
* github
* lnmp(optional)

    KeluLinuxKit.sh init
