# KeluLinuxKit
a tools to setup your dev environment on Debian. 

## Waht's in it
* oh my zsh
* Maximum Awesome 
* tmux-powerline
* iptables
* openresty
* php5.6
* postgresql9.4
* composer
* docker
* docker_pptp
* docker_shadowsocks
* l2tp
* server-client pptp & shadowsocks

## Install

    ./keluLinuxKit.sh install all
    ./keluLinuxKit.sh run all
