# KeluLinuxKit
a tools to setup your dev environment on Debian. 

## What's in it
* oh my zsh
* Maximum Awesome 
* tmux-powerline
* iptables
* openresty
* php7.0
* php-composer
* postgresql9.4
* docker
* docker_pptp
* docker_shadowsocks
* l2tp

## Install

    ./keluLinuxKit.sh install all
    ./keluLinuxKit.sh install lnmp
    ./keluLinuxKit.sh run all
